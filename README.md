# Workforce Intelligence Dashboard

A SQL project that answers 6 real HR business questions using PostgreSQL.

## Business Questions Answered
1. Which employees are flight risks?
2. Is there a gender pay gap?
3. Are top performers being paid fairly?
4. Which departments have the highest attrition?
5. How does tenure distribute across teams?
6. Did we lose any high performers?

## Key Findings
- Sales had a 41.7% attrition rate — highest in the company
- Engineering had a 10.3% gender pay gap
- 5 top performers (all female) are underpaid by up to 11.7%
- 0 high performers lost yet — but that's a warning, not a win

## Tools Used
PostgreSQL 18 | pgAdmin 4

## Files
- `workforce_intelligence.sql` — all queries with comments
- `employees.csv` — simulated dataset (50 employees)
